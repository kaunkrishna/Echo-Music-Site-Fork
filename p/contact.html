Before Contacting Us

To help us help you faster, please check these resources first:

  • README: Check our README file for installation and setup instructions
  • GitHub Issues: Search existing issues to see if your problem has been reported
  • Documentation: Check our setup guide for detailed documentation

What to Include in Your Message

When contacting us, please include:

  • Device Information: Android version, device model
  • App Version: Version of Echo Music you're using
  • Problem Description: Clear description of the issue
  • Steps to Reproduce: How to reproduce the problem (for bugs)
  • Screenshots/Logs: If applicable, include relevant screenshots or log files

Response Times

I aim to respond to all inquiries within:

  • Email: 24-48 hours
  • GitHub Issues: 2-3 days
  • Bug Reports: Priority response within 24 hours
  • Feature Requests: Community discussion and evaluation
